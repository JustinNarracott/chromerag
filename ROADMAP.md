# ChromeRAG Roadmap

ChromeRAG is an experiment in browser-native, local-first AI.

## V1 - Local URL RAG

- Localhost web app
- URL ingestion
- text extraction
- chunking and retrieval
- Chrome Prompt API synthesis

## V2 - Local enterprise files

- Local folder indexing
- SharePoint / OneDrive sync folder support
- PDF and DOCX extraction
- skipped-file handling
- reset / refresh controls

## V3 - Product polish

- incremental refresh
- changed-file detection
- source filters
- Markdown / HTML export
- clickable evidence links

## V4 - Workspaces

- named workspaces
- source viewer
- research workflows
- strict evidence mode
- prompt budgeting

## V5 - Connectors

- simplified UI
- connector abstraction
- Microsoft Graph / SharePoint connector direction
- local + URL + file sources
- cleaner exports

## V6 - Chrome extension

- Chrome side panel
- index current page
- optional auto-indexing while browsing
- query browser-captured evidence locally

## Potential V7 - Agentic research worker

Possible but not necessarily the main product direction.

- browser-controlled research runs
- seed URLs and goal
- bounded navigation
- local evidence store
- generated report

## Current priority

The strongest direction is not a generic local agent. It is a private, browser-native intelligence layer for existing work:

- local files
- SharePoint / OneDrive
- browser tabs
- internal docs
- technical documentation
- research sessions
